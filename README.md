# Смысловой поиск по фото

Поиск по картинкам по смыслу запроса, а не по названию файла или тегам. Пишешь текстом — «я на пляже прошлым летом», «пёс в снегу», «диван как этот, но серый» — и получаешь релевантные фото. Всё считается локально, картинки в облако не уходят.

## Зачем

Когда в архиве десятки тысяч снимков, нужное вручную не найдёшь. Поиск по имени файла и тегам не работает, потому что не знает, что изображено на картинке. Плюс языковой барьер: запрос на русском, а описания в датасетах на английском.

Целимся в две аудитории. Обычный пользователь с личным архивом на 10–50 тыс. фото — ему нужна одна строка поиска и сетка результатов, без настроек. И бизнес (маркетплейсы, фотостоки, медиа-архивы, модерация) — там нужен движок для каталога, API и контроль над качеством поиска.

## Как устроено

Каждая картинка и текстовый запрос переводятся в вектор (эмбеддинг) моделями CLIP и BLIP, дальше поиск идёт по векторам через FAISS. Русский запрос переводится на английский локально через Opus-MT, так что язык ввода не важен.

Основное:

- Модели CLIP / BLIP / BLIP-2, взаимозаменяемы, можно собирать в ансамбль (вес w=0.75)
- Тройной ансамбль BLIP-2 даёт +4.5 к Recall@1
- Точный режим — реранжирование через ITM, медленнее, но точнее
- Инференс на CPU, без видеокарты; GPU нужен только для тяжёлого точного режима и индексации больших каталогов
- Индекс на FAISS, поэтому поиск не проседает на большой базе

Данные для обучения и замеров — публичный датасет MS COCO (~330 тыс. изображений, лицензия CC BY 4.0).

## Метрики

- Recall@1 = 57.6%
- Время поиска < 1 сек на CPU
- Качество ранжирования меряем по MRR, mAP и Median Rank

## Функции

Что есть и что в работе:

- Текстовый поиск на русском
- Поиск «картинка + текст» («как на этом фото, но зимой»)
- Негатив — исключить лишнее («но без людей»)
- Поиск по вырезанному фрагменту картинки
- Точный режим (ITM-реранж)
- Heatmap — подсветка, по какой области нашлось фото
- Мультиязычность (локальный русский + другие языки через API)
- Batch-загрузка каталога и API для встраивания
- Шеринг результата ссылкой
